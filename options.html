◈ CLAUDE USAGE — SETTINGS

ORG ID

Your Claude org UUID — needed to poll the usage endpoint. Find it in DevTools → Network → any /api/organizations/{uuid}/ request.

Paste once and forget. Stored locally, never leaves your browser except in requests to claude.ai.

When enabled, overrides window.fetch on claude.ai to sniff the org ID from the first API call. Automatically disables itself once the ID is found and stored.

UTILIZATION GOALS

Warn when utilization exceeds these thresholds. Goal markers appear on bars.

NOTIFICATIONS

IOT PUSH

Push credentials directly to your NodeMCU over LAN. Session cookie is read automatically from your browser — no manual copy-paste needed.

IP shown on the OLED display after boot.
Keeps the device in sync when your session cookie rotates. Recommended.

DATA

Usage history is stored locally in your browser. 30-day rolling window.
